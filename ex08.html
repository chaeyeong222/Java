1. head 태그 
   - 메타데이터(데이터에 대한 데이터)의 컨테이너,
   - html 태그의 자식
   - 메타데이터 : 문서 제목, 
2. head 태그 자식 태그
   ㄱ. title 태그 : 문서제목, 브라우저의 제목 표시줄 또는 페이지 탭 표시.
   ㄴ. style 태그: html문서의 스타일 정보 정의
   ㄷ. link 태그: 외부 리소스 연결, 외부 css 파일(자주 사용)
   ㄹ. meta 태그 :
      1) meta charset="UTF-8" 웹 문서에 사용된 인코딩(문자셋) 정의
      2) 키워드 정의 meta name="keywords" content="html,css,javascript"
        왜? 검색엔진에 대한 키워드 정의
   ㅁ. script 태그
   ㅅ. base 태그: 페이지의 모든 상대 주소에 대한 기본 주소 및 대상을 지정하는 태그
