    br, hr, meta, img 필요x
  4.img scr=""alt="" 등호 주위에 공백 사용x
  5. p  /p 꼭 닫기태그 해주기

     요청 url -> 웹 서버(톰캣) : 대소문자 구분

     외부 css 파일 :       .css
     외부 javascript 파일: .js
