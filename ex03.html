[뷰포트(Viewport)]
 1. 웹 페이지에서 사용자가 볼 수 있는 영역
 2. 표시 영역은 디바이스 장치에 따라 화면의 크리가 다르다.
 3. 테이블, 스마트폰, 컴퓨터화면 - 뷰포트 다름
   여러 장치에서 브라우저를 사용해서 웹 페이지를 장치 화면에 맞게 축소확대
 4. 장치의 크기 배율에 맞게 웹 페이지의 크기 제어
  뷰포트 설정
   
   width=device-width : 장치의 화면 너비에 따라 웹 페이지 너비 설정
   initial-scale=1.0 : 브라우저에서 페이지 처음 로드할 때 초기 확대/축소 수준 설정
